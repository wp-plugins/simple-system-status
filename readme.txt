View Information about your WordPress Configurationa and Server Information that is useful for debugging.

== Description ==

Simple System Status adds a page under the 'tools' section of your WordPress installation that will allow you to
quickly and easily view the details of your current WordPress Installation from: WordPress non-sensitive details, Server Information, PHP Details, Theme Details, Actived and Deactivated Plugins.

You can use this for your own debugging purposes or for sharing with Support via two methods: A Dynamic Link, or a text file you can download and send.

== Installation ==

Installing "Simple System Status" can be done either by searching for "Simple System Status" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
